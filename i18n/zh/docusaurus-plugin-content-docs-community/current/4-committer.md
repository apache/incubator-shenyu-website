## 提交者提名

当你做了很多贡献以后，社区会进行提名。
成为committer你会拥有

* Apache ShenYu仓库写的权限
* [jetbrains正版全家桶](https://www.jetbrains.com/shop/eform/apache)

## 提交者责任

- 解答issue问题。
- 指导贡献者加入社区。

## 评审Pull Request

* 尽可能给出合理的建议。

* pull request 应该标有 `shenyu Lable` 并且必须设置进度表。

* 合并完成后，需要检查关联的 `issueNo` 是否关闭。
